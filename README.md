# Unifor General Services Manager

## About the Project

This is a Full Stack Web Application developed for the Web Development course at the University of Fortaleza (Unifor).

The application helps manage the university's general services team. It organizes staff schedules, assigns cleaning zones (classrooms, labs, blocks), and controls the inventory of cleaning supplies.

**Note:** Since this was an academic project, the application interface is entirely in Portuguese.
